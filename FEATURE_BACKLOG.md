# Let's Snog - Feature Backlog

This document captures suggested next-phase features after the base product launch.

## 1) High-Impact Product Features

### 1. Voice Intro Clips (15s max)
- Let users add a short voice note to profile.
- Improves authenticity and profile quality.
- Helps reduce low-effort or fake accounts.

### 2. "Tonight / This Week" Intent Toggle
- Users indicate immediate dating intent.
- Improves relevance of daily matches.
- Increases match-to-date conversion.

### 3. Double-Date Mode
- Users can opt in with a friend and match with another pair.
- Reduces first-date anxiety.
- Can improve retention for new users.

### 4. Post-Date Follow-Up Automation
- If both users submit positive feedback, suggest follow-up plans automatically.
- Offer 3 smart date suggestions and one-tap "propose next date".
- Keeps momentum and reduces drop-off after first date.

### 5. Chat Reply Prompts
- Lightweight AI prompts in chat ("ask about X", "suggest Y place").
- Helps users who stall in conversation.
- Reduces dead chats.

## 2) Safety and Trust Features

### 1. Photo Verification
- Selfie/gesture-based verification flow.
- Add verified badge to profile.

### 2. Enhanced Reporting
- Report flow with optional screenshot/media evidence upload.
- Better moderation signal quality.

### 3. Block Controls
- Support soft block, hard block, and hide preferences.
- Gives users finer control over feed and interactions.

### 4. Date Safety Check-In
- Optional timer-based check-in for planned dates.
- Push reminder and emergency contact flow.

## 3) Monetization Features

### 1. Premium Lite
- Add limited paid tier with:
  - +5 daily matches
  - advanced filters
- Keep core free experience strong.

### 2. Boost Windows
- Time-boxed profile visibility boost (e.g., 30 minutes).
- Monetizes intent during peak windows.

### 3. Readiness Badge
- Optional paid/earned "Ready this week" signal.
- Improves matching for active daters.

## 4) Growth and Operations Features

### 1. Referral Codes
- Invite flow with referral attribution and reward logic.
- Adds low-cost growth loop.

### 2. Event Waitlist + Auto-Fill
- Queue users when events are full.
- Auto-promote on cancellations.

### 3. Weekly Dating Digest
- Weekly summary email/push:
  - new matches
  - replies
  - date conversion status

### 4. Moderation Admin Queue
- Admin dashboard queue for reports with priority scoring.
- Faster moderation response and lower user risk.

## Suggested Implementation Order

1. Safety baseline: photo verification + improved reporting
2. Conversion: intent toggle + chat prompts
3. Retention: follow-up automation + weekly digest
4. Monetization: premium lite + boosts
5. Expansion: double-date mode + advanced moderation tooling
